# 🎓 SGU Schedule Bot

Telegram-бот для получения расписания занятий СГУ (Саратовский государственный университет).

## Возможности

- 📅 Расписание любой группы по факультету и форме обучения
- 🔆 Расписание на сегодня одной кнопкой
- 👨‍🏫 Поиск расписания преподавателя по фамилии / ФИО
- 👤 Профиль пользователя (сохраняется в SQLite, не пропадает при перезапуске)
- 💡 Быстрый ввод: просто напишите номер группы — бот найдёт её на вашем факультете
- 📋 История поиска
- ⚡ Кэш расписаний (1 час) и списка преподавателей (6 часов)

## Структура

```
SheduleBot/
  bot.py           # точка входа
  config.py        # токен, факультеты, константы
  database.py      # SQLite: профили, кэш, история
  fetcher.py       # HTTP + кэш расписаний
  parser.py        # парсинг HTML таблиц СГУ
  keyboards.py     # все клавиатуры Telegram
  utils.py         # вспомогательные функции
  requirements.txt
  handlers/
    menu.py        # /start, помощь, история
    profile.py     # настройка профиля
    schedule.py    # расписание групп
    teacher.py     # поиск преподавателей
```

## Установка и запуск

```bash
git clone https://github.com/MaximGaranin/SheduleBot.git
cd SheduleBot
pip install -r requirements.txt
```

Вставьте токен бота в `config.py`:
```python
BOT_TOKEN = "ВАШ_ТОКЕН"
```

Запуск:
```bash
python bot.py
```

База данных создаётся автоматически в папке `data/sgu_bot.db` при первом запуске.

## Таблицы БД

| Таблица | Содержимое | TTL |
|---|---|---|
| `profiles` | Факультет, форма, группа пользователя | навсегда |
| `schedule_cache` | HTML страниц расписания | 1 час |
| `teacher_cache` | Результаты поиска преподавателей | 6 часов |
| `search_history` | История поиска пользователя | навсегда |
